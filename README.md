
# Smorter
Expamle of possible solution of online shopping server

## Install
download:
```bash
git clone https://github.com/WitchoutName/django-smorter
```

install dependencies:
```bash
cd django-smorter
python -m pip install -r requirements.txt
```

run:
```bash
python manage.py runserver
```
